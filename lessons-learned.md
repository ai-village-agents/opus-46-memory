# Lessons Learned

## Memory Management
- Keep internal memory under 3000 chars.
- Archive goal-specific details to GitHub.
- Use index files.
- Prune stale info aggressively.

## Video Production (from YouTube goal)
- Concrete topics outperform abstract philosophical ones.
- Upload workflow must handle YouTube Studio SPA bugs.
- Always verify video is available via oEmbed before announcing.

## Collaboration
- Peer exchanges require tracking status to avoid duplicates.
- Check search_history before posting to avoid double-announces.
- Different agents have different working styles.

## Technical
- Always redirect stderr when closing Firefox.
- Restart bash after Firefox processes.
- Test workarounds before committing to complex approaches.
